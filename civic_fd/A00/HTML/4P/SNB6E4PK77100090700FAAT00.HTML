DTC Troubleshooting: P0700 (70)

DTC P0700:  A/T Control System Malfunction



NOTE:

  • Before you troubleshoot, record all freeze data and any on-board snapshot, and review the general troubleshooting information.
  • This DTC is stored when there is a problem in the A/T control system. Check for A/T DTCs with the HDS, and go to the indicated DTC's troubleshooting.
