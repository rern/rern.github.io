A/T 1st, 2nd, and 3rd Clutch Reassembly

1.Soak the clutch discs throughly in ATF for a minimum of 30 minutes.

2.Install the waved spring (A) in the 1st clutch drum (B). Starting with the clutch waved-plate, alternately install the clutch waved-plates (C) (4), and discs (D) (4), and install the clutch end-plate (E) with the flat side down on the top disc.

 

3.Install the waved spring (A) in the 2nd clutch drum (B). Install the clutch flat-plate (C), and clutch disc (D). Starting with the clutch waved-plate, alternately install the clutch waved-plates (E) (3), and discs (F) (3), and install the clutch end-plate (G) with the flat side down on the top disc.

 

4.Install the waved spring (A) in the 3rd clutch drum (B). Install the clutch flat-plate (C), and clutch disc (D). Starting with the clutch waved-plate, alternately install the clutch waved-plates (E) (3), and discs (F) (3), and install the clutch end-plate (G) with the flat side down on the top disc.

 

5.Install the snap ring with a screwdriver to secure the clutch-end-plate.

 

6.Check that the clutch piston moves by applying air pressure into fluid passage.
