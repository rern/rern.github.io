DTC Troubleshooting: U0122 (107-4)

DTC U0122:  Lost Communication with VSA Modulator Control Unit


NOTE: Before you troubleshoot, record all freeze data and any on-board snapshot, and review General Troubleshooting Information.

1.Clear the DTC with the HDS.

2.Turn the ignition switch ON (II).

3.Check that DTC U0122 recurs.

Is DTC U0122 indicated?


YES - Go to 4.


NO - Intermittent failure, the system is OK at this time.n


4.Check whether any DTC is indicated in the DTCs/Freeze Data in VSA Mode Menu with the HDS.

Is any DTC indicated in the VSA?


YES - Troubleshoot for indicated DTC in the VSA.


NO - Go to 5.


5.Turn the ignition switch OFF.

6.Jump the SCS line with the HDS.

7.Disconnect PCM connector A (44P).

8.Disconnect the VSA modulator-control unit 47P connector.

9.Check for continuity between PCM connector terminal A36 and VSA modulator-control unit 47P connector terminal No. 14, and between A37 and terminal No. 30.

 

Is there continuity?


YES - Update the PCM if it does not have the latest software, or substitute a known-good PCM, then recheck. If the symptom/indication goes away with a known-good PCM, replace the original  PCM.n


NO - Go to 10.


10.Check the communication line between the VSA modulator-control unit and gauge control module by the HDS.

Is there malfunction in the F-CAN line between the VSA modulator-control unit and gauge control module?


YES - Repair open in the wires between VSA modulator-control unit 47P connector and CAN communication lines.n


NO - Substitute a known-good VSA modulator-control unit, then recheck. If the symptom/indication goes away with a known-good VSA modulator-control unit, replace the original  VSA modulator-control unit.n
