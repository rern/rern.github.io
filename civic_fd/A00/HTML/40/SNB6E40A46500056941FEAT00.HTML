Starter Solenoid Test

1.Disconnect the negative cable from the battery first, then disconnect the positive cable.

2.Remove the intake manifold.

3.Disconnect the S and M connectors from the starter solenoid.

 

4.Check the hold-in coil for continuity between the S terminal and the armature housing (ground). There should be continuity.

  • If there is continuity, go to 5.
  • If there is no continuity, replace the solenoid.

5.Check the pull-in coil for continuity between the S terminal and M terminal. There should be continuity.

  • If there is continuity, the solenoid is OK.
  • If there is no continuity, replace the solenoid.

6.Install the intake manifold.

7.Connect the positive cable to the battery first, then connect the negative cable.

8.Set the clock.

9.Do the power window control unit reset procedure.
