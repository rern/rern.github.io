M/T Reverse Lockout System Description

At a vehicle speed of 20 km/h (12 mph) or more, a signal from the vehicle speed sensor (VSS) activates the reverse lockout solenoid, which pushes the select lock cam into the locked position. As a result, the select lever cannot rotate to the reverse select position, making it impossible to engage reverse gear. At a vehicle speed of 15 km/h (9 mph) or less, the signal from the VSS is interrupted which turns off the reverse lockout solenoid. The select lock return spring pulls the select lock cam back, enabling the select lever to move freely so that reverse gear can be selected.

Above 20 km/h (12 mph) ON Not allowed
