DTC Troubleshooting: 11-01

DTC 11-01:  Low/High Voltage for IG1 (Initial diagnosis)


1.Clear the DTC using the HDS.

2.Turn the ignition switch OFF.

3.Start the engine.

4.Wait at least 60 seconds.

Does the EPS indicator come on?


YES - Go to 5.


NO - Check for loose wires or poor connections. If the connections are good, the system is OK at this time.n


5.Verify the DTC.

Is DTC 11-01 indicated?


YES - Check for a poor connection at the battery terminal or battery. If they are good, go to the Alternator and Regulator Circuit Troubleshooting: R18A1 engine model, K20Z2 engine model.n


NO - Perform the appropriate troubleshooting for the code indicated.n
