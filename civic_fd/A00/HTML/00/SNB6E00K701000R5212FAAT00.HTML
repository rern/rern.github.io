DTC Troubleshooting: 52-12

DTC 52-12:  Motor Stuck OFF


1.Turn the ignition switch ON (II).

2.Clear the DTC with the HDS.

3.Turn the ignition switch OFF, then turn it ON (II) again.

4.Operate each SOLENOID in the ABS FUNCTION TEST with the HDS. Repeat this 5 times for each solenoid.

5.Check for DTCs with the HDS.

Is DTC 52-12 indicated?


YES - Check for loose terminals in the ABS modulator-control unit 25P connector. If necessary, substitute a known-good ABS modulator-control unit and retest: Except K20Z2 engine model, K20Z2 engine model.n


NO - Intermittent failure, the system is OK at this time. Check for loose terminals at the ABS modulator-control unit 25P connector. Refer to intermittent failures troubleshooting.n
