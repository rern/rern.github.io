SRS Symptom Troubleshooting - SRS indicator stays on, but no DTCs are stored

SRS indicator stays on, but no DTCs are stored


1.Turn the ignition switch ON (II), and see if the PGM-FI indicator come on.

Does the PGM-FI indicator come on?


YES - Go to 2.


NO - Go to the PGM-FI system troubleshooting:n

  • K20Z2 engine
  • Except K20Z2 engine


2.Connect the HDS to the DLC.

3.Turn the ignition switch ON (II).

4.Make sure the HDS communicates with the vehicle and the SRS unit. If it does not, troubleshoot the DLC circuit:

  • K20Z2 engine
  • Except K20Z2 engine

5.In Body Electrical on the HDS, check for DTCs.

Is DTC B1187 indicated?


YES - Go to the troubleshooting DTC B1187.


NO - Faulty gauge control module (tach); replace the gauge control module.n
