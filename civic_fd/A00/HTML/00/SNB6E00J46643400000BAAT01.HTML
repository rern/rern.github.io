Keyless/Power Door Locks/Security System Symptom Troubleshooting Index - Related to the door locks

Related to the door locks


If the door lock system works properly, but the keyless operation only does not work, refer to the keyless entry part (except power door locks).

NOTE: The door lock system does not function when:
  • Any of the doors are open.
  • Ignition key is in the ignition.
  • Ignition switch is ON (II).

No. Symptom Tests 
1 The door(s) will not lock or unlock. 
  • MICU input test
  • Driver's door key cylinder switch test
  • Door lock switch test
  • Door lock knob switch test
 
2 The door(s) will not lock. 
  • MICU input test
  • Door switch test
  • Door lock knob switch test
 
3 The door(s) will not unlock. 
  • MICU input test
  • Door switch test
  • Door lock knob switch test
 
4 Door(s) will relock even if the door is opened once after unlock operation. Driver's door switch test 
5 Door(s) will lock and unlock even if the ignition key is in the ignition switch. 
  • Body Electrical DTC check
  • Ignition key switch test
 
6 Door(s) will lock and unlock even if the ignition switch is ON (II). Ignition switch test 
7 Sometimes door(s) will not lock and unlock. 
  • Door switch test
  • Driver's door key cylinder switch test
  • Loose or poor connection of the door lock system composed parts connector
