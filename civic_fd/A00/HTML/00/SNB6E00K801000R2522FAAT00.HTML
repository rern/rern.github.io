DTC Troubleshooting: 25-22

DTC 25-22:  Yaw Rate Sensor Stuck

1.Turn the ignition switch ON (II).

2.Clear the DTC with the HDS.

3.Do the VSA sensor neutral position memorization.

4.Turn the ignition switch OFF, then turn it ON (II) again.

5.Test-drive the vehicle. Drive the vehicle at 10 km/h (7 mph) or more.

6.Check for DTCs with the HDS.

Is DTC 25-22 indicated?


YES - Go to 7.


NO - If any DTCs are indicated, go to the indicated DTCs troubleshooting. If DTCs are not indicated, intermittent failure, the system is OK at this time. Check for loose terminals between the yaw rate-lateral acceleration sensor 5P connector and the VSA modulator-control unit 37P connector. Refer to intermittent failures troubleshooting.n


7.Check the LATERAL ACCELERATION SENSOR and YAW RATE S in the VSA DATA LIST with the HDS.

Are there 0 m/s indicated?


YES - Go to 8.


NO - Do the VSA sensor neutral position memorization, then go to 1 and recheck.n


8.Test-drive the vehicle. Check the YAW RATE S in the VSA DATA LIST with the HDS while driving in corners.

Does the indicated value change?


YES - Intermittent failure, the system is OK at this time. Check for loose terminals between the yaw rate-lateral acceleration sensor 5P connector and the VSA modulator-control unit 37P connector. Refer to intermittent failures troubleshooting.n


NO - Replace the yaw rate-lateral acceleration sensor.n
