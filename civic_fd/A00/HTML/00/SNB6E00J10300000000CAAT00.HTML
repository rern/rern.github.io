Audio System Description

Overview


The audio unit acts as the ‘‘processor'' for all audio functions. Select audio functions from the audio unit, the audio remote (on steering wheel). The audio display provides the current audio status.

The cassette player unit signal to the audio unit. In addition communicates with the audio unit via the GA-Net bus.

A security signal is daisy-chained between the audio and vehicle components for integration into the vehicle's security system.
