DTC Troubleshooting: 25-11, 25-12, 25-13, 25-16, 25-21

DTC 25-11:  Yaw Rate Sensor Internal Circuit Malfunction (Initial)
  
  
DTC 25-13:  Yaw Rate Sensor Communication Error
  
  
DTC 25-21:  Yaw Rate Sensor Neutral Position Malfunction


1.Turn the ignition switch ON (II).

2.Clear the DTC with the HDS.

3.Do the VSA sensor neutral position memorization.

4.Turn the ignition switch OFF, then turn it ON (II) again.

5.Check for DTCs with the HDS.

Is DTC 25-11, 25-12, 25-13, 25-16, or 25-21 indicated?


YES - Replace the yaw rate-lateral acceleration sensor.n


NO - Intermittent failure, the system is OK at this time. Check for loose terminals between the yaw rate-lateral acceleration sensor 5P connector and the VSA modulator-control unit 37P connector. Refer to intermittent failures troubleshooting.n
