Keyless/Power Door Locks/Security System Symptom Troubleshooting - Keyless UNLOCK/LOCK operation troubleshooting

Keyless UNLOCK/LOCK operation troubleshooting


1.Turn the ignition switch OFF.

2.Remove the ignition key from the ignition switch.

3.Close and lock the doors.

4.Try to unlock the doors with the keyless transmitter.

Do the door lock actuators work normally?


YES - The system is OK at this time.n


NO - Go to 5.


5.Connect the HDS to the data link connector.

6.Turn the ignition switch ON (II).

7.Enter the BODY ELECTRICAL, then select the KEYLESS.

8.Do the KEYLESS CHECK.

Is the KEYLESS ENTRY TRANSMITTER CODE RECEIVED indicated?


YES - Recheck the system again. If the UNLOCK/LOCK operation only does not work, replace the under-dash fuse/relay box assembly.n


NO - Recheck the system again. If the UNLOCK/LOCK operation only does not work, and HDS communicates correctly, replace the  immobilizer-keyless control unit.n
