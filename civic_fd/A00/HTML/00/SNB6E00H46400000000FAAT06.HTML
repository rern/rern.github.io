SRS Symptom Troubleshooting - Faulty seat belt reminder light

Faulty seat belt reminder light


1.Observe these precautions:

  • Unbuckle the left side seat belt.
  • Unbuckle the right side seat belt.
  • Make sure nothing is on or under the seat.

2.Connect the HDS to the DLC located under driver's side of the dashboard.

3.Turn the ignition switch ON (II).

4.Make sure the HDS communicates with the vehicle and the SRS unit. If it does not, troubleshoot the DLC circuit:

  • K20Z2 engine
  • Except K20Z2 engine

5.From the Main Menu on the HDS, select SRS, then PARAMETER INFORMATION. In the PARAMETER INFORMATION Menu, select ‘‘SRS PSS'', and read the HDS indicated.

Is ‘‘OFF'' indicated?


YES - Go to 8.


NO - Go to 6.


6.Disconnect the floor wire harness 2P connector (A) from the front passenger's seat sitting switch.

 

7.From the Main Menu on the HDS, select SRS, then PARAMETER INFORMATION. In the PARAMETER INFORMATION Menu, select ‘‘SRS PSS'', and read the HDS indicated.

Is ‘‘ON'' indicated?


YES - Short to ground in the floor wire harness; replace the floor wire harness.n


NO - Faulty front passenger's seat sitting switch.n


8.Observe these precautions:

  • Unbuckle the left side seat belt.
  • Unbuckle the right side seat belt.
  • I make it the state that people more than 45 kg in weight sit down on in front passenger's seat.

9.From the Main Menu on the HDS, select SRS, then PARAMETER INFORMATION. In the PARAMETER INFORMATION Menu, select ‘‘SRS PSS'', and read the HDS indicated.

Is ‘‘ON'' indicated?


YES - Go to 10.


NO - Go to 6.

10.Check the seat belt reminder light.

Does the seat belt reminder light stay on?


YES - Faulty gauge control module.n


NO - Faulty SRS unit or poor connection at SRS unit connector B (28P) and the SRS unit or poor connection at front passenger's seat belt sitting switch 2P connector and front passenger's seat sitting switch. Check the connection; if connection is OK and DTC is not indicated, replace the SRS unit.n


11.Disconnect the floor wire harness 2P connector (A) from the front passenger's seat sitting switch.

 

12.From the Main Menu on the HDS, select SRS, then PARAMETER INFORMATION. In the PARAMETER INFORMATION Menu, select ‘‘SRS PSS'', and read the HDS indicated.

Is ‘‘ON'' indicated?


YES - Short to ground in the floor wire harness; replace the floor wire harness.n


NO - Faulty front passenger's seat sitting switch.n

13.Disconnect the floor wire harness 2P connector (A) from the front passenger's seat sitting switch.

 

14.Install a jumper wire between the No. 1 terminal and the No. 2 terminal of the floor wire harness 2P connector.

15.From the Main Menu on the HDS, select SRS, then PARAMETER INFORMATION. In the PARAMETER INFORMATION Menu, select ‘‘SRS PSS'', and read the HDS indicated.

Is ‘‘ON'' indicated?


YES - Faulty front passenger's seat sitting switch.n


NO - Open in the floor wire harness; replace the floor wire harness.n
