Power Window Master Switch Test/Replacement

1.Carefully pry out the master switch (A).

 

 

2.Disconnect the 20P connector (B) from the power window master switch.

3.Check for continuity between the terminals in each switch position according to the tables.

Driver's Switch

The driver's switch is combined with the control unit so you cannot isolate the switch to test it. Instead, run the master switch input test procedures 3, 4, 15, and 20. If the tests are normal, the driver's switch must be faulty. Replace the switch.
