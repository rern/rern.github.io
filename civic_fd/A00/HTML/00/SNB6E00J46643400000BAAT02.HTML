Related to the keyless entry system (except power door locks):


The keyless control unit and keyless receiver are combined with the immobilizer control unit. If the power door lock system works properly, but there is a malfunction when you operate the keyless entry system, check the immobilizer system and keyless entry system by the order of the priority shown:

Order of Priority Symptom Possible cause 
1 Immobilizer indicator blinks after the ignition switch turned ON (II). Immobilizer indicator blinks troubleshooting. 
2 Engine does not start with the immobilizer key. Engine does not start with the immobilizer key troubleshooting. 
3 Immobilizer indicator does not come on. Check the MIL indication.
  • If the MIL comes on, go to the PGM-FI System MIL circuit troubleshooting.
    • 1.8 L
    • 2.0 L
  • If the MIL does not come on, go to the gauge control module self-diagnostic function.
 
4 Immobilizer indicator dose not go off. Immobilizer indicator does not go off troubleshooting. 
5 Keyless operation dose not function at all (LOCK, UNLOCK). keyless operation troubleshooting. 
6 The doors can unlock with the transmitter, but cannot lock. Keyless LOCK/UNLOCK operation troubleshooting. 
7 The doors can lock with the transmitter, but cannot unlock. Keyless LOCK/UNLOCK operation troubleshooting. 
8 The doors will relock even if the doors are opened once after unlocking with the transmitter. The doors will relock even if the doors are opened once after unlocking with the transmitter troubleshooting. 
9 The doors will lock and unlock with the transmitter, but the horn does not sound and the lighting system does not flash. The doors will lock and unlock with the transmitter, but the horn does not sound and the lighting system does not blink.
  • Horn Test.
  • MICU Input Test.
 
10 Keyless operation will work even the ignition switch is ON (II). Keyless operation will work even the ignition switch is ON (II).
  • MICU Input Test.
  • Ignition Switch Test.
 
11 Keyless operation will work even the ignition key is in the ignition switch. Ignition key switch test.
