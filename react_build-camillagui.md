React - Build `camillagui`
---
Require minimum 2GB RAM
- Install
```sh
curl -L https://github.com/HEnquist/camillagui/archive/refs/tags/v1.0.0.tar.gz | bsdtar xf -
cd camillagui-1.0.0
pacman -Sy npm
npm install reactjs
```

- Build
	- Edit
	- Build: `npm run build`
	- Result files in `build` directory
